Quelques consignes générales

  • faire les points dans l'ordre et ne passer au point suivant que lorsque le précédent est achevé et testé,
  • utiliser la console autant que nécessaire Pour bien observer le contenu et le type des différentes variables,
  • écrire du code structuré et correctement indenté,
  • écrire du code largement documenté !
  • l'ajout du code JavaScript est fait entre les balises <script>...<script> de ce fichiet html

Eléments de départ fournis quel que soit l'exercice :

  • ce fichier (soutien_xx.html) qui est un conteneur de l'exercice
  • un fichier de style CSS (style.css) déjà chargé via le header du document html
  • un fichier Javascript (tools.js), déjà chargé via le header du document html, contenant divers outils :
    • une fonction d'affichage formatté d'un tableau (Array) à la fin du document html courant,
    • une fonction d'affichage formatté d'un résultat à la fin du document html courant,
    • une fonction de génération aléatoire d'un nombre entier compris entre 2 valeurs,
    • une fonction de génération d'une série aléatoire de nombres entiers rangés dans un tableau de type Array,
    L'utilisation de ces outils n'est pas systématiquement nécessaire (cela dépend des exercices). Par contre, vous n'avez, en aucun cas, besoin :
    • de modifier le code de ces outils,
    • de copier le code des fonctions contenues dans tools.js dans votre propre code.
      Si vous utilisez les fonctions outils, vous ne pouvez que les "appeler".

Soutien-2 #06 - Synthèse

En s'inspirant de la réflexion et du code mis en oeuvre lors des 4 exercices précédents (Soutien-2 #02 à #05) réaliser les points suivants :

  1. Récupérer la fonction cube depuis l'exercice Soutien-2 #05. Elle sera utilisée telle que ; inutile de la modifier.
  2. Écrire une fonction generateSrcData qui construit un tableau de valeurs aléatoires.
    Cette fonction est appelée par appui sur le bouton "Generate".
    Pour construire le tableau, generateSrcData utilise la fonction randomArrayOfIntFromInterval mise à disposition dans tools.js.
    Les paramètres de construction du tableau sont acquis à travers le formulaire d'entrée:
    • nombre d'éléments de la série
    • valeur minimum de la série générée
    • valeur maximum de la série générée
    Le tableau généré par generateSrcData est stocké dans une variable qui devra être partagée avec les autres fonctions du code.
  3. Écrire la fonction go qui s'exécute lors de l'appui sur le bouton "Go !".
    Cette fonction exploite le tableau des valeurs "sources" qui auront été générées auparavant (par l'appui sur le bouton "Generate").
    Cette fonction a pour tâche de construire un nouveau tableau qui contient la transformation du tableau "source" par la fonction cube. C'est à dire que le tableau "résultat" ainsi construit contient les données du tableau "source", une à une transformées par la fonction cube.

    Le tableau "résultat" sera restitué à l'utilisateur en utilisant la fonction fournie displayArray.
